Demo

I have a little demo of one of my projects called photoquads

It takes an image and, using a quad tree, it will split the image into rectangles depending on which node in the tree is the most farthest from it's estimated average color.

It was originally written in Java but then I re-wrote it in javscript to create the web demo. Hope you enjoy!
